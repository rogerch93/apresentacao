Skills

HTML5

HTML uma linguagem de marcação de hipertexto, um bloco básico mais não menos importante para desenvolvimento de apps ou simples websites pela web. Esta ferramenta utiliza suas tags para exibir imagens, textos e também se conectar a outros sites atraves de seus links.

CSS3

CSS "Folha de cascata" uma linguagem de estilo usada para dar "vida" ao HTML estilizando textos com cores, imagens com sombras ou reflexos, podemos usalo até mesmo para fazer algumas animações basta ter imaginação, em pouco tempo você consegue fazer algumas animações bem estilizadas. Com o avanço da tecnologia, hoje em dia o CSS é capaz de fazer muitas coisas referenciado a estilização

Js

JavaScript uma linguagem leve, interpretada e baseada em objetos. Mais connhecida como a linguagem de script para páginas web, mas não se engane ela também pode ser utilizada em ambientes sem browser. Com o Node.Js é possivel construir um back-end de uma WebAplication utilizando JavaScript, fazendo o uso dela tanto no back-end como front-end.

C#

O C# é uma linguagem de programação moderna, orientada a objeto e de tipo seguro. O C# permite que os desenvolvedores criem muitos tipos de aplicativos seguros e robustos que são executados no .NET. O C# tem suas raízes na família de linguagens C.
